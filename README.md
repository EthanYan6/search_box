# search_box
实现一个类似于百度的搜索框，前期只添加前端代码，后期尝试对接公众号进行搜索。

## use

执行如下命令进行下载：

```shell
git clone https://github.com/EthanYan6/search_box.git
```

然后打开如下页面即可使用：

```shell
search_box\template\html\index.html
```

> 如果要使用查询功能，需要联网
